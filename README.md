# LEET 추리논증

LEET 추리논증 문제를 한 화면에 한 문항씩 풀고, 선택 직후 정답과 구조화된 해설을 확인하는 정적 웹앱입니다.

## 구성

- 한 문항 단위의 앞면/해설면 플립 인터페이스
- 데스크톱 2단 구성과 모바일 통합형 문항 카드
- 미응답 상태의 해설 열람과 선지 재클릭 선택 해제
- 연도 선택, 문항 바로가기, 로컬 진도 저장
- 논증 흐름·비교표·계산식 등 HTML 해설 시각화
- 빌드 과정과 외부 런타임 의존성이 없는 GitHub Pages용 정적 파일

## 로컬 실행

ES 모듈을 사용하므로 파일을 직접 열지 말고 간단한 정적 서버로 실행합니다.

```bash
python3 -m http.server 4174
```

그다음 `http://localhost:4174`를 엽니다.

## 데이터 검증

```bash
node scripts/validate-data.mjs
```

문항 번호, 선택지 수, 정답 범위, 공식 정답표, 조합형 선지와 해설의 O/X 일치 여부를 검사합니다.

## 새 학년도 추가

1. `data/2027.js`를 참고해 `data/2028.js` 같은 새 파일을 만듭니다.
2. 각 문항에 `id`, `type`, `stem`, `sections`, `statements`, `choices`, `answer`, `explanation`을 작성합니다.
3. `data/index.js`에서 새 파일을 가져와 `exams` 배열에 추가합니다.
4. 필요하면 검증 스크립트에 해당 학년도 공식 정답표를 추가합니다.

`data/helpers.js`의 `section`, `table`, `flow`, `visualTable`, `equations`를 이용하면 문제 표와 해설 시각화를 같은 형식으로 추가할 수 있습니다.

## 출처

문제 저작권은 법학전문대학원협의회에 있습니다. 이 저장소는 학습 편의를 위해 사용자가 제공한 2027학년도 추리논증 자료를 HTML 데이터로 구성한 비공식 학습 도구입니다.
